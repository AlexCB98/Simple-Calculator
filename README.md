# Python Calculator 🧮

A simple interactive calculator built in Python as a learning project to practice functions, dictionaries, loops, and recursion.

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Power (`**`)
- Modulo (`%`)
- Floor division (`//`)

## How it works

1. The user enters a number
2. Chooses an operation
3. Enters another number
4. The result is calculated and displayed
5. The user can:
   - Continue calculating with the result
   - Start a new calculation
   - Exit the program

## Extra Features

- Input validation for invalid operations
- Clean restart using recursion
- Simple user interface with ASCII logo
- Clear terminal output between calculations

## Run the project

```bash
python main.py
```
